# 👕 JQG Studio - Loja Virtual de Roupas

## 📖 Sobre o Projeto

O JQG Studio é uma loja virtual fictícia de roupas desenvolvida como Projeto Interdisciplinar do 1º semestre do curso de Desenvolvimento de Software Multiplataforma da Fatec Itaquera.

O objetivo da atividade foi aplicar os conhecimentos iniciais de desenvolvimento web por meio da criação de um site funcional, contendo navegação entre páginas e organização de conteúdo, simulando a presença digital de uma marca de moda.

## 🚀 Funcionalidades

* Página inicial (Home)
* Página de vendas/produtos
* Menu de navegação entre páginas
* Apresentação da marca
* Exibição de produtos
* Layout organizado e intuitivo
* Estrutura responsiva para diferentes dispositivos

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## 🎯 Objetivos do Projeto

* Aplicar conceitos fundamentais de desenvolvimento web.
* Desenvolver páginas utilizando HTML e CSS.
* Implementar navegação entre múltiplas páginas.
* Criar uma interface amigável para o usuário.
* Simular um ambiente de comércio eletrônico.

## 🏠 Página Inicial

A página Home apresenta a identidade da marca JQG Studio, destacando sua proposta, estilo e principais produtos, além de fornecer acesso rápido às demais áreas do site.

## 🛍️ Página de Vendas

A página de vendas exibe os produtos da loja, permitindo que os usuários visualizem as peças disponíveis e conheçam melhor o catálogo da marca.
